# Python Beginner Projects Collection

This repository contains a collection of 5 beginner-friendly Python applications. Each project is designed to demonstrate fundamental programming concepts, data handling, and simple game logic in Python. It is inspired by tutorial posted on youtube.com.

## Projects Overview

1. **[Project 01: Slot Machine](./project_01)**
   A terminal-based slot machine game. Place bets, spin the wheels, and see if you win!

2. **[Project 02: Quiz Game](./project_02)**
   A command-line quiz application that dynamically loads questions from a JSON file and tracks your score.

3. **[Project 03: To-Do App](./project_03)**
   A simple To-Do list manager. Add, complete, and delete tasks, with all data saved persistently using JSON.

4. **[Project 04: Contact Management](./project_04)**
   A Contact Management System to add, search, and delete contacts, demonstrating CRUD operations with a JSON backend.

5. **[Project 05: Tic-Tac-Toe](./project_05)**
   The classic game of Tic-Tac-Toe, featuring both a simple AI and an unbeatable Minimax algorithm AI.

## Getting Started

To run any of the projects, ensure you have Python 3 installed. Navigate to the specific project directory and run the main Python file. For example:

```bash
cd project_01
python main.py
```
