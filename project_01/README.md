# Project 01: Slot Machine

A simple terminal-based slot machine game written in Python. This project demonstrates loops, conditions, random choice generation, and basic user input validation.

## Features
- Dynamic board generation based on probabilities.
- Betting system with multiple lines.
- Win detection and balance tracking.

## How to Play
Run the `main.py` script:
```bash
python main.py
```
You will be prompted to enter a starting balance, the number of lines to bet on, and the bet amount per line. Press 'Enter' to spin or 'q' to quit.
