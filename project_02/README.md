# Project 02: Quiz Game

A command-line quiz application that tests your knowledge across various topics. Questions are dynamically loaded from a `questions.json` file.

## Features
- JSON-based question storage for easy extension and modification.
- Random selection of questions.
- Score tracking at the end of the quiz.

## How to Play
Run the `main.py` script:
```bash
python main.py
```
You will be asked how many questions you want to answer. Choose the correct option by typing its corresponding number.
