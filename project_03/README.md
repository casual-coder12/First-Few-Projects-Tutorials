# Project 03: To-Do App

A simple command-line To-Do list manager. This application allows you to add, complete, and delete tasks.

## Features
- Persistent storage using JSON.
- Status tracking for tasks (Completed vs Pending).
- Interactive menu.

## How to Run
Run the `main.py` script:
```bash
python main.py
```
Follow the on-screen prompts to manage your tasks.
