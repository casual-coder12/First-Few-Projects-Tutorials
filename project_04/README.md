# Project 04: Contact Management

A command-line application for managing a list of contacts. Contacts are stored in a `contacts.json` file.

## Features
- Add new contacts (Name, Age, Email).
- Search for contacts by name.
- Delete existing contacts.
- Persistent data storage.

## How to Run
Run the `main.py` script:
```bash
python main.py
```
Follow the interactive prompts to manage your address book.
