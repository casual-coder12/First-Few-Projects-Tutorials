# Project 05: Tic-Tac-Toe

A terminal-based Tic-Tac-Toe game. This project includes two versions of the game, showcasing different AI difficulty levels.

## Versions
- `main.py`: Features a basic, somewhat random AI. Good for testing and casual play.
- `main_minmax.py`: Features an unbeatable AI powered by the Minimax algorithm.

## How to Play
Run one of the scripts:
```bash
python main_minmax.py
```
Or for the simpler AI:
```bash
python main.py
```
Follow the on-screen prompts to place your "X" or "O" on the board.
